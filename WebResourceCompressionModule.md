# Configuration #

Example Configuration:
```
<configuration>
	<system.web>
		<httpModules>
			<add name="WebResourceCompressionModule" type="Talifun.Web.Compress.WebResourceCompressionModule, Talifun.Web"/>
		</httpModules>
	</system.web>
</configuration>
```
